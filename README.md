# Invoice Frontend

## Tech Stack
- React, TypeScript, Redux Toolkit, TailwindCSS

## Setup

1. Clone repository:- https://github.com/MuhammadAnasRasheed/Invoice-Management-System-frontend.git
2. Run `npm install`
3. Create `.env` file with `VITE_API_URL=your_backend_url`
4. Run `npm run dev`

## Features
- User authentication
- Customer management
- Invoice creation
- Search invoices and customers
